# EduGap Excalidraw Diagrams

Each chart is split into a separate Excalidraw file so it can be edited and exported independently.

Rendered SVG and PNG exports are available in `docs/diagram/images/`.

## Files

- `01-system-runtime-topology.excalidraw` - layered system runtime topology.
- `02-auth-role-gate.excalidraw` - auth, BFF token forwarding, JWT verification, and role gate swimlanes.
- `03-socratic-ai-chat-runtime.excalidraw` - Socratic chat runtime, LangGraph, RAG branch, reflection loop, and observability.
- `04-rag-ingestion-retrieval.excalidraw` - offline slide ingestion and online pgvector retrieval.
- `05-adaptive-recommendation.excalidraw` - adaptive recommendation decision system.
- `06-adaptive-submit-transaction.excalidraw` - replay-resistant submit path and Supabase RPC transaction.
- `07-algorithm-internals.excalidraw` - BKT, Elo, LinUCB, ZPD reward, and Sherman-Morrison update.
- `08-async-calibration-outbox.excalidraw` - calibration outbox and background worker.
- `09-concept-graph-bitemporal-mastery.excalidraw` - concept graph propagation and bitemporal mastery history.
- `10-onboarding-diagnostic.excalidraw` - first-run diagnostic and personalization baseline.
- `11-core-data-model.excalidraw` - clustered ERD-level data model.
- `12-cicd-operations.excalidraw` - CI/CD, deploy, health, and manual verification flow.
