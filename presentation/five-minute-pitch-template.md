# Five-Minute Pitch Template

## Slide 1: Title & The Hook (0:00 - 0:45)
- **Visual**: Slide showing target user in their environment.
- **Talking Points**: Hook the audience with the specific pain point. Outline the negative business/social impact.

## Slide 2: The Core Problem & Baseline (0:45 - 1:30)
- **Visual**: Flowchart contrasting current process vs. proposed shortcut.
- **Talking Points**: Present baseline comparison. Introduce target user KPIs.

## Slide 3: The Solution & Live Demo (1:30 - 2:45)
- **Visual**: Live Workspace screen capture or screen share.
- **Talking Points**: Walk through uploading the input file, configuring options, running the capability, and showing the output artifact. Highlight dynamic parameter rendering.

## Slide 4: System Architecture & Security (2:45 - 3:45)
- **Visual**: Component flowchart (Frontend Next.js, Backend FastAPI, local storage).
- **Talking Points**: Explain modular design (optional modules disabled). Detail security shields (path traversal block, log redaction).

## Slide 5: Evaluation Reports & Metrics (3:45 - 4:30)
- **Visual**: Metrics dashboard (success rates, duration histogram, error logs count).
- **Talking Points**: Present data profiling stats. Walk through metric registry results.

## Slide 6: Summary & Future Vision (4:30 - 5:00)
- **Visual**: Key takeaway quotes.
- **Talking Points**: Pilot launch plans. Scaling steps. Concluding statement.
