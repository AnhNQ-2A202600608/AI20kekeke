# Live Demo Script Template

Follow this script step-by-step during the presentation to showcase the workspace interface.

## Step 1: Open the Application
- Navigate to `http://localhost:3000` (next.js app).
- Show the **Home** dashboard outlining starter features.

## Step 2: Navigate to Workspace
- Click on **Workspace** in the sidebar.
- Point out the capability dropdown selection list (only enabled capabilities show up).

## Step 3: Show Dynamic Parameters
- Select the active capability.
- Show how the UI dynamically renders input fields (checkboxes, textareas, number slots) matching the backend parameter schema specification.

## Step 4: File Upload & Run
- Click "Choose File" and upload a sample file.
- Highlight the successful upload notification showing the system-generated file ID.
- Click **Run Capability**.
- Point out the active processing status indicator.

## Step 5: Render Artifacts
- Show the execution output card containing: Run ID, started/completed timestamps, and duration in milliseconds.
- Point out the **Grounded Artifact Preview** rendering the output report content.
- Briefly navigate to **Results** or **Runs** pages to show how execution traces are cataloged in system storage.
